Agent Skill

surge-deploy

Deploy any static build to the web in seconds. One command. No config.

Install
npx skills add aruntemme/surge-deploy

How it works

  1. Checks surge is installed and authenticated
  2. Detects your build folder automatically
  3. Handles SPA routing (creates 200.html)
  4. Deploys to surge.sh
  5. Saves domain in .surge-domain for re-deploys

Trigger with

  • /static-deploy
  • deploy to surge
  • static deploy this project

Frameworks

React Vite Vue Next.js Astro SvelteKit Hugo 11ty HTML

Anything that produces a build folder.

Features

Domain Memory
Remembers URL per project via .surge-domain
Preflight
Validates auth, build, index.html before deploy
SPA Support
Auto-creates 200.html for client-side routing
CI/CD
GitHub Actions and GitLab CI templates included

Compatible Agents

Claude Code, Cursor, GitHub Copilot, Codex, Windsurf, Cline, Amp, Goose, OpenCode, Roo Code, and any agent supporting the Agent Skillsstandard.
